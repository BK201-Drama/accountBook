# 钱多多记账



## 下载方式：

项目中的**apk-final**文件夹可以下载文件包

apk-final文件夹里面有my-app文件夹，该文件夹为在HBuilderX环境下开发运行的app文件，apk文件也是从该文件夹打包而来



## 技术栈：

前端：使用react框架进行开发，选择的组件库是React-vant

后端：采用传统的ssm框架
